# Norwaymaxxing legal site

Static Privacy Policy and Terms for Norwaymaxxing. The site has no runtime dependencies, cookies, forms, or client-side JavaScript.

## Deploy with Vercel

1. Import the `norwaymaxxing` repository in Vercel.
2. Set **Root Directory** to `legal-site`.
3. Leave Framework Preset as **Other**.
4. Leave Build Command and Output Directory empty.
5. Deploy and assign the final domain.
6. Set `EXPO_PUBLIC_LEGAL_BASE_URL` in the app to the deployed origin, without a trailing slash.

Expected routes:

- `/privacy`
- `/terms`

Before release, have a qualified lawyer review the documents and add a public support email when one is available. Until then, privacy requests can be sent to the controller's published postal address.
